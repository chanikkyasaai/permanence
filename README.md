# PERMANENCE

PERMANENCE is a reinforcement-learning environment and tooling stack for modeling irreversible corporate decision dynamics, cascade lockouts, and catastrophe-aware policy behavior.

## Included Components

- Core PERMANENCE environment package
- Training and evaluation scripts
- Warmup trace generation pipeline
- Interactive judge sandbox
- Dashboard backend and React UI
- Ghost demo exporter and playback mode

## Quick Start

1. Install dependencies for Python and dashboard frontend.
2. Run training artifacts and dashboard backend.
3. Use ghost playback mode for offline demo stability.
